-Overview

This game is called Cave. The ship wants to go as deep as possible. When it hits either the above or bottom part of the wall, the game is over. Possible commands are to go up or to do nothing(because the ship is under gravity, it goes down naturally). In general, the command is inputted by a human, but pre-learned DQN deals with the command in this game. The commands are an output of its DQN.

-How to play

  1. Press the train button, which may take a few minutes to learn
  2. Press the start button
  3. See how it works
